Tema: Etiquetas Básicas de HTML

HTML es un lenguaje de marcado, esto quiere decir que se escribe usando codigos y que no se necesita compilarlo para que una persona lo pueda leer, estos codigos sirven para dar instrucciones al navegador web sobre como debe mostrarse lo que esta escrito dentro de el.
Las etiquetas en HTML contienen palabras clave, atributos y valores rodeados de los signos <,>

Por ejemplo:
html lang="es" donde html es la palabra clave, lang es el atributo a quien le corresponde el valor es
La mayoria de las etiquetas se utilizan en pares, es decir una etiqueta de apertura y una de cierre. Se escriben de la siguiente manera:

  • El caracter de apertura <
  • El nombre de la etiqueta que se va a usar
  • El nombre del atributo con su respectivo valor (nombre="valor ")
  • El caracter de cierre >

¿Cómo funcionan las etiquetas HTML?

Una etiqueta le indica al navegador dónde empieza y dónde termina cada elemento, mientras que un atributo describe las características.
Los elementos HTML se dividen en tres partes, las cuales crean un elemento HTML.

  • El caracter de apertura, es decir, donde comienza el elemento, encerrado asi: <>
  • El contenido que ve el cliente
  • El caracter de cierre, es igual que el de apertura solo que lleva una barra inclidada.
